# フォームについて

## as
フォームオブジェクトの`as`オプションで送信するハッシュのキー名を指定できる。

## f.select
- セレクトボックスを生成する
- プロパティ名は送信するハッシュのキー名。通常はカラム（属性）名にすることが多い。
-  **HTMLオプションを使う場合は、無印オプションを省略できない。** かならず空の大かっこを入れること。
- 選択肢は配列もしくはハッシュで
```erb
<%= f.select( プロパティ名, 選択肢, {オプション}, {HTMLオプション} )
```
```rb
# 選択肢が配列の場合
[[ 表示名1, 値1 ], [ 表示名2, 値2 ], ...]

# 選択肢がハッシュの場合
{ 表示名1: 値1, 表示名2: 値2, ... }
```
